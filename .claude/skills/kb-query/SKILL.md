# KB Query

Answer a research question using the knowledge base wiki.

Follow the **Query** workflow in `knowledge_base/SCHEMA.md`. That file is the single source of truth for page templates, naming conventions, and workflow steps.

## Guidelines

- Always start from `index.md` — do not grep wiki files unless the index is insufficient
- Answers can take different forms: narrative, comparison table, bullet list, or a full synthesis page
- If the wiki lacks information to answer the question, say so explicitly and suggest sources to look for
- Good queries that reveal gaps are valuable — flag them for future ingest
