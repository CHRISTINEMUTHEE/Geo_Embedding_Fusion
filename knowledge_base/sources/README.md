# Sources

Immutable raw inputs. Drop files here, then run `/kb-ingest`. See `SCHEMA.md` for naming conventions.

## Contents

- `papers/` — Research papers
- `articles/` — Blog posts, tutorials, reports
- `notes/` — Personal and meeting notes
- `assets/` — Images, diagrams, data files

Large binary files (PDFs, images) are git-ignored. See `.gitignore`.
